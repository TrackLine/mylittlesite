Политика обработки персональных данных

SafeWayVPN — ваша конфиденциальность под защитой

1. Общие положения

Настоящая политика определяет порядок обработки и защиты персональных данных пользователей сервиса SafeWayVPN, предоставляемого через Telegram-бота @SafeWayVPN_bot.

2. Какие данные мы собираем

  • Telegram ID пользователя
  • Техническая информация, необходимая для поддержки (например, тип устройства, версия приложения)

SafeWayVPN не хранит логи вашей интернет-активности, не отслеживает и не анализирует ваш трафик.

3. Цели обработки данных

  • Идентификация пользователя для предоставления доступа к сервису
  • Техническая поддержка и решение вопросов пользователей
  • Обеспечение безопасности и предотвращение нарушений

4. Передача и хранение данных

Ваши данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством. Все данные хранятся только в объёме, необходимом для работы сервиса и поддержки пользователей.

5. Безопасность

SafeWayVPN применяет современные технические и организационные меры для защиты ваших данных от несанкционированного доступа, изменения или уничтожения.

6. Права пользователя

  • Получать информацию о своих данных
  • Требовать их удаления или исправления
  • Отозвать согласие на обработку данных (при этом доступ к сервису может быть ограничен)

Для реализации своих прав вы можете обратиться в поддержку через Telegram: @ormnet_support_bot или по email: support@saratov.live.

7. Изменения политики

SafeWayVPN может обновлять настоящую политику без предварительного уведомления. Актуальная версия всегда доступна на сайте.
